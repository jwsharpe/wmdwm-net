Before you use this

This workbench is a tool for validating plaintext guesses against Eye Message isomorphs (gap-pattern shape). Matching shapes never prove a reading; mismatched shapes can falsify one.

This site is >95% AI-generated. All of the UI was created by a machine; simple analytic code was also created by machine. The isomorph generation was reviewed but also initially created by AI.

Version — this notice returns after every update.

Leaving strict mode

Validation is at its best in strict mode, where typing stays on isomorphs, shared columns, and gap sites — the places plaintext claims are actually constrained.

With strict off you can fill arbitrary cells. The checker still runs, but false positives show up more often (compatible-looking repeats that aren’t real structure). Prefer keeping strict on unless you know why you need free typing.

Plaintext / falsify guesses against gap-pattern shape

GAK is perfectly isomorphic: equal plaintext must land on equal ciphertext gap patterns. Type a candidate on the grid — repeated words that disagree are impossible. Supporting lenses (isomorphs, shared columns, gaps) show where repeats already live in the ciphertext so you know what to test; they don’t replace the check.
